Bu projede Java dili kullanılarak video üzerinde hareket eden nesnelerin kırmızıya boyanması, ayrıca arkaplan fotoğrafının
çıkarılması amaçlanmıştır. Java dili ile video framelere bölünüp hareketli pixeller kırmızıya boyanmıştır.
Python dili ile geliştirilen programda ise kırmızıya boyanan fotoğraflar tekrar bir video haline getirilmiştir.
